# مسیر (Masir)

وب‌اپلیکیشن استاتیک و آمادهٔ پروداکشن برای دانش‌آموزان دبیرستان ایران که برای **امتحانات نهایی** و **کنکور** برنامه می‌چینند.

بدون بک‌اند، بدون لاگین، بدون دیتابیس. داده فقط در `LocalStorage` دستگاه ذخیره می‌شود.

## امکانات

- تایم‌لاین روزانه با بلوک‌های دقیق مطالعه / تست / آزمون / تحلیل / استراحت
- موتور قانون‌محور بر اساس پایه، رشته، وضعیت امتحان، سطح و نسبت تشریحی/تستی
- حالت اجرا با تایمر عادی و پومودورو
- استراحت کوتاه/بلند قابل تنظیم
- موسیقی آرام (Web Audio، بدون فایل صوتی)
- تیک فعالیت‌ها و درصد پیشرفت روز
- خروجی تصویر PNG و چاپ
- PWA با Service Worker و نصب روی موبایل
- RTL کامل و ریسپانسیو

## سناریوهای پوشش‌داده‌شده

| محور | مقادیر |
|------|--------|
| پایه / رشته | یازدهم تجربی، یازدهم ریاضی، دوازدهم تجربی، دوازدهم ریاضی |
| وضعیت امتحان | عادی، تعویق‌شده، نامشخص |
| سطح | ضعیف، متوسط، قوی |
| نسبت | ۸۰/۲۰ تا ۴۰/۶۰ تشریحی به تستی |

بلوک‌های تخصصی شامل: آزمون جامع ماز، خیلی سبز، گزینه‌دو، مدارس برتر، تحلیل آزمون، برگشت از پاسخنامه به کتاب، حفظیات شیمی، خلاصه‌نویسی گیاهی، پیش‌خوانی دوازدهم، جبران پایه دهم.

## ساختار

```
css/           سیستم دیزاین و استایل
js/data/       کاتالوگ دروس و بلوک‌ها
js/engine/     قوانین، پلنر، سناریوها
js/components/ تایمر، موسیقی، اکسپورت
js/core/       LocalStorage
tests/         تست ماتریس سناریوها
sw.js          Service Worker
manifest.json  PWA manifest
```

## اجرا

هر سرور استاتیک کافی است (به خاطر ES modules و Service Worker نباید با `file://` باز شود):

```bash
npm start
# یا: python3 -m http.server 4173
```

سپس `http://127.0.0.1:4173` را باز کنید.

## تست

```bash
npm test
```

ماتریس کامل سناریوها (۱۸۰ حالت) را اجرا می‌کند.

## معماری موتور

1. ورودی پروفایل از ویزارد
2. `computePolicy()` قوانین را شلیک می‌کند (نسبت مؤثر، جبران پایه، آزمون‌ها، …)
3. `buildDailyPlan()` بودجهٔ زمانی را بین تشریحی/تستی تقسیم و بلوک‌ها را زمان‌بندی می‌کند
4. وضعیت تیک‌ها در LocalStorage با `instanceId` پایدار می‌ماند

## مجوز

پروژهٔ آموزشی — استفاده آزاد برای دانش‌آموزان و مشاوران.
