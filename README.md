# مسیر (Masir)

وب‌اپلیکیشن **استاتیک** برای دانش‌آموزان یازدهم و دوازدهم — وقتی امتحان‌ها هی جابه‌جا و تعویق می‌شوند.

بدون بک‌اند، بدون لاگین، بدون دیتابیس. داده فقط در `LocalStorage` دستگاه ذخیره می‌شود.

## جریان دانش‌آموز

1. انتخاب پایه (یازدهم / دوازدهم) و رشته
2. **امتحان بعدیت چیه؟** — انتخاب از لیست دروس
3. وضعیت خبر:
   - کنسل / تعویق شده
   - هنوز خبر رسمی‌ای نیومده
4. اگر کنسل شده:
   - «این درس رو ضعیفم» یا «می‌تونم ببندمش»
   - در حالت ضعیف: آیا امتحان برگزارشدنی بعدی را هم ضعیفی؟
5. انتخاب پیشنهادها (صبح تا ظهر / ظهر تا عصر / عصر تا شب — بدون ساعت‌بندی)
6. اجرا با تایمر، تیک، خروجی تصویر

## امکانات

- برنامهٔ سیال بر اساس آخرین تصمیم و انتخاب‌ها
- فقط سه بازهٔ روز؛ ساعت دقیق نمی‌بندد
- لحن خودمونی مربی‌محور
- بلوک‌ها: آزمون جامع ماز / خیلی‌سبز، مدارس برتر، تحلیل، پاسخنامه→کتاب، حفظیات شیمی، گیاهی
- تایمر عادی و پومودورو، موسیقی آرام، PWA آفلاین

## اجرا

```bash
npm start
# یا: python3 -m http.server 4173
```

سپس `http://127.0.0.1:4173` را باز کنید (به‌خاطر ES modules با `file://` باز نکنید).

## تست

```bash
npm test
```

ماتریس مسیرهای مشاوره‌ای (پایه × رشته × وضعیت خبر × سطح × امتحان برگزارشدنی) را اجرا می‌کند.

## ساختار

```
css/           سیستم دیزاین
js/data/       دروس، بلوک‌ها، جریان پیشنهادها (flow)
js/engine/     پلنر و سناریوها
js/components/ تایمر، موسیقی، اکسپورت
js/core/       LocalStorage
tests/         تست ماتریس
```
